# WATI Landing Page Clone

## 🚀 Overview
This project is a frontend implementation of the WATI landing page using HTML, CSS, and JavaScript.

## 🛠️ Tech Stack
- HTML5
- CSS3 (BEM methodology)
- Vanilla JavaScript

## 📱 Features
- Fully responsive design
- BEM naming convention
- Pixel-approximate UI
- Functional navigation

## ⚙️ Setup
1. Clone repo
2. Open index.html in browser

## 💡 Approach
- Broke UI into reusable blocks
- Used mobile-first design
- Focused on spacing and typography accuracy
